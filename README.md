# **Projeto Final**
## IPVC - ESTG - ECGM
### Realizado por: Francisco Pimenta n25418


### **Resumo**

  Este projeto foi realizado no âmbito da disciplina de Projeto do Curso ECGM no IPVC. Consiste num website sobre o IPVC, mais concretamente escolas, eventos e cursos nas mesmas. Tem também a utilização de objetos 3D, movimentados pelo utilizador e automáticamente usando o código. Linguagens utilizadas são:
  + JavaScript
  + PHP
  + CSS
  + SCSS
  + HTML


  Para a parte 3D uso duas bibliotecas JS:
  + THREE.js
  + GSAP.js

  
### **JavaScript**

  Usado para interações dentro do Website, exemplo:
  + Clicar num botão adiciona/remove uma classe de um certo elemento
  + Linguagem base a modelação 3D existente

### **PHPP**
  Comunicação com a base de dados:
  + Busca de Informação
  + Criação de Informação
  + Edição de Informação
  + Destruição de Informação

### **HTML**
  "Linguagem" usada para a visualização do Website da parte do Utilziador, não há mais que se possa dizer

### **CSS E SCSS**
  Styling do HTML do website
