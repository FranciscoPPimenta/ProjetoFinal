# **Projeto Final**
## IPVC - ESTG - ECGM
### Realizado por: Francisco Pimenta n25418


### **Resumo**

  Este projeto foi realizado no âmbito da disciplina de Projeto do Curso ECGM no IPVC. Consiste num website sobre o IPVC, mais concretamente escolas, eventos e cursos nas mesmas. Tem também a utilização de objetos 3D, movimentados pelo utilizador e automáticamente usando o código. Linguagens utilizadas são:
  + JavaScript
  + PHP
  + CSS
  + SCSS
  + HTML


  Para a parte 3D uso duas bibliotecas JS:
  + THREE.js
  + GSAP.js
